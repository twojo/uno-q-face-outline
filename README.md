# Greetings from Arduino UNO Q Face Outline Tracker Demo 👋

> _Transform your Arduino UNO Q into a polite digital companion that recognizes you and waves hello whenever you approach your desk, using face recognition models and a servo motor._

An app for Arduino UNO Q which uses face recognition models and a servo motor to greet people

Track the outline of faces detected and show emojies of different expressions made and for how long

👉 Arduino ProjectHub Tutorial [here](https://projecthub.arduino.cc/leocavagnis/greetings-from-arduino-uno-q-6a403b).

## Troubleshooting
If you have issue due Servo library please do the following:
```bash
arduino-cli lib update-index
arduino-cli lib install Servo
```
